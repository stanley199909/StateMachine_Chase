## 設計思想：StateMachineを軸にした明確な状態管理

本作『スタートマシンチェイス』では、特に以下の点を重視して設計・実装しています：

- **敵AI**：各状態（巡回・追跡・攻撃・逃走など）を明確なStateクラスに分割  
- **ゲーム進行管理**：シーン遷移やフェーズ管理もStateMachineで一元化  
→ 処理の流れが追いやすく、後から見ても「何が起こっているか」が一目瞭然になるよう心がけました

個人制作とチーム制作の両方を経験する中で、「動くこと」以上に**「後から読んでも理解しやすいコード」**の価値を実感しました。  
そのため、クラス設計・フォルダ構成も「役割ごとに明確に分離する」ことを徹底しています。

このプロジェクトは、そうした考え方を1本のゲームとして形にしたものです。  
今後もStateMachineを中心とした設計力の向上を目指し、もっと読みやすく・拡張しやすいコードを書いていきたいと考えています。

操作方法
ゲームプレイはキーボード
WASDを使って移動します、
そしてEnterキーを使用して各シーンへ転移します
